# Malika-WEBSITE

Malika Masarrath – Personal AI-Powered Productivity Dashboard https://malika-masarrath.github.io/FOCUS-FLOW/
This repository contains my fully personalized productivity dashboard, designed to support and gamify my journey toward becoming a Full Stack Generative AI Engineer. Built using HTML, CSS, and JavaScript in Visual Studio Code, with all code generated by GitHub Copilot, this project is tailored exclusively for my personal use—my name, Malika, is integrated throughout the interface, including headings and motivational quotes.
🔹 Key Features:
- Gamified Progress System: Dopamine meter, adrenaline challenges, quest chain road, XP tracker, streak counter, and progress percentage.
- Time Management Tools: Focus timer, stopwatch with lap tracking, sound effects, and visual feedback.
- Hydration Tracker: Daily water intake logging with streak tracking and reset functionality.
- Motivation Center: Personalized inspirational quotes with sound and visual effects; quotes can be added as tasks.
- To-Do List: Task entry with productivity levels (low, medium, high), completion tracking, and daily goal progress.
- Calendar View: Full-year calendar interface with motivational overlays (non-live, under development).
- Mini Games: Includes memory game, quick reaction, pattern match, and adrenaline rush; tracks completions and best scores.
- Analytics & Reports: Auto-generated daily, weekly, monthly, and annual performance summaries.
- Reward & Badge System: Tracks earned badges and rewards (some features under development).
- Settings & Customization:
- Audio: Enable/disable sound effects.
- Visuals: Toggle particles, animations, glitter.
- Accessibility: High contrast mode, reduced motion, dyslexia-friendly font.
- Data Management: Export, import, and reset dashboard data.
🛠️ Development Status:
This dashboard is a work in progress. Some features are incomplete, and several areas require refinement to improve practicality and efficiency. I’m actively working to expand functionality, fix bugs, and enhance the overall experience.
